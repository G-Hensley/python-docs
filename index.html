Introduction to Python

What is Python?

Python is a high-level, interpreted programming language known for its readability and simplicity. It supports multiple programming paradigms and is widely used for web development, data science, automation, and more.

Variables & Data Types

Variables

In Python, you don't need to declare a variable's type. Just assign a value to it.

Examples:


    x = 10  # Integer
    name = "John"  # String
    pi = 3.14  # Float

Common Data Types

  • int: Integer numbers (e.g., 5, 100, -45)
  • float: Decimal numbers (e.g., 3.14, -0.001)
  • str: Strings (e.g., "Hello", "Python")
  • bool: Boolean values (True or False)

Basic Input and Output

Printing to the Console

Use the print() function to display output in Python.

Example:

print("Hello, world!")

Control Flow

If Statements

Conditional statements allow you to execute code based on conditions.

Example:


    age = 18
    if age >= 18:
        print("You are an adult.")
    else:
        print("You are a minor.")

Functions

Defining a Function

Functions allow you to group code that performs a specific task.

Example:


    def greet(name):
    print("Hello, " + name)
                
    greet("Alice")

Lists

Creating a List

Lists in Python can store multiple items in a single variable.

Example:


    fruits = ["apple", "banana", "cherry"]
    print(fruits[0])  # Output: apple

Other List Operations:

  • append(): Add an element to the end of the list
  • remove(): Remove a specific element from the list
  • sort(): Sort the elements of the list in ascending order
  • pop(): Remove the last element of the list
  • len(): Return the length (number of elements) of the list
